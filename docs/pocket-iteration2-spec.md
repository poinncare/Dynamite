## 1. Локализация настроек + выбор языка
- Сделать двуязычную локализацию EN/RU для ВСЕХ строк окна настроек (SettingsView и его секций) и всех строк UI вкладки Clipboard (контекстные меню, тултипы, подписи).
- Механика: enum `AppLanguage { english, russian }` (Defaults.Serializable), ключ `appLanguage` в Defaults.Keys, **по умолчанию английский**; `en.lproj`/`ru.lproj` Localizable.strings + хелпер `L(_ key:)`, который резолвит строку из бандла выбранного языка (Bundle(path: forResource: lang)). Не менять системный AppleLanguages.
- В настройках добавить пикер «Language / Язык»: English | Русский. Переключение применяется к настройкам и вкладке Clipboard сразу или после переоткрытия окна настроек (сразу — предпочтительно; если сразу нереализуемо без большой перестройки — допустимо после переоткрытия, отметить в отчёте).
- Русские переводы писать самому, аккуратные и краткие.

## 2. Хоткей = toggle
- Cmd+Shift+C (clipboardHistoryPanel): если шторка закрыта — открыть на вкладке clipboard; если шторка УЖЕ открыта на вкладке clipboard — закрыть (`vm.close()`). Если открыта на другой вкладке — переключить на clipboard, не закрывая. Обработчик: `boringNotchApp.swift:413-440`.
- Убрать/не применять автозакрытие через 3с для этого хоткея, если оно мешает (как у toggleNotchOpen App:396-404 — проверить, не наследуется ли).

## 3. Пробел = попап полного текста (как в Pasta)
- При нажатии Space на выделенной карточке с текстовым контентом (string/rtf/html) показать поверх шторки окошко-попап с ПОЛНЫМ текстом: скруглённый тёмный контейнер, скроллируемый текст, максимум ~80% ширины/высоты шторки.
- Закрытие: повторный Space, Escape или клик вне попапа. Пока попап открыт, WASD/hjkl/стрелки не двигают выделение.
- Для нетекстовых карточек Space ничего не делает (или показывает увеличенный thumbnail для картинок — опционально, не обязательно).

## 4. Убрать поиск полностью
- Удалить поле поиска из вкладки Clipboard и всю связанную логику фокуса (isSearchFocused, passthrough-ветки в ClipboardKeyboardMonitor). Лента карточек занимает освободившуюся высоту.
- `ClipboardSearch.swift` и fuse-зависимость можно оставить неиспользуемыми или удалить из target — на выбор, отметить в отчёте.

## 5. Таймер времени
- Переместить в ПРАВЫЙ нижний угол карточки.
- Формат огрублённый, без секунд: `<60с → "now"`, затем минуты `1m…59m`, затем часы `1h…23h`, затем дни `1d…`. Локализация не нужна (латиницей, как в Pasta).
- Обновление лейблов периодически (таймер ~30с на вью), чтобы "now" превращалось в "1m" без переоткрытия.

## 6. Футер карточки
- Иконка приложения-источника — в НИЖНИЙ ЛЕВЫЙ угол (сейчас сверху). Символ типа контента (doc-иконку) убрать совсем.
- Строку-заголовок сверху заменить на многострочное превью текста с самого начала контента (как в Pasta): текст занимает тело карточки сверху вниз, lineLimit по высоте карточки.

## 7. Навигация: WASD + vim
- Убедиться, что W/A/S/D работают идентично стрелкам (сейчас заявлено, но проверить после удаления поиска).
- Добавить vim-клавиши: h = влево, l = вправо, j = вниз, k = вверх (в однорядной ленте j/k = как S/W). Все три раскладки (стрелки, WASD, hjkl) — один и тот же обработчик перемещения.

## 8. Режим Command (как в Pasta)
- Пока удерживается ⌘ (flagsChanged в ClipboardKeyboardMonitor): в правом нижнем углу каждой видимой карточки вместо таймера показывается `⌘1`, `⌘2`, … по порядку карточек слева направо. Отпустил ⌘ — вернулся таймер.
- Нажатие ⌘+цифра (1–9) при открытой вкладке = вставить (copy+paste) соответствующую карточку.
