### Poll tick → item
`checkForChangesInPasteboard()` (`Clipboard.swift:151-231`):
1. Skip if `changeCount` unchanged.
2. Skip if `Defaults[.ignoreEvents]` (one-shot via `ignoreOnlyNextEvent`).
3. Skip if pasteboard types should be ignored (`shouldIgnore(types)` — autoGenerated/concealed/transient + custom ignored types).
4. Skip if source app ignored (`shouldIgnore(bundleId)`).
5. Merge all `pasteboardItems` into one `[HistoryItemContent]` (handles multi-item copy from BBEdit/Edge).
6. Create `HistoryItem(contents:)`, set `application` + `title`, call hooks.

### Supported / ignored pasteboard types
- **Supported (hardcoded set)** `Clipboard.swift:19-26`: `.fileURL`, `.html`, `.png`, `.rtf`, `.string`, `.tiff`
- **Enabled by default** = all of files+images+text (`Defaults.Keys+Names.swift:4-8, 17-18`): fileURL, png, tiff, html, rtf, string
- **Always ignored type markers** `Clipboard.swift:27-31`: autoGenerated, concealed, transient (+ user `ignoredPasteboardTypes`)
- Filters: strip `dyn.*`, `com.microsoft.ole.source.*`; special MS Word link handling (`Clipboard.swift:199-209`)
- Extra display types on model (not all polled as “enabled”): heic, jpeg, universalClipboard (`HistoryItem.swift:150-199`, `NSPasteboard.PasteboardType+Types.swift`)

## 4) Paste into frontmost app

### Flow
1. User selects item → `History.select` (`History.swift:300-334`)
2. `Clipboard.copy(item)` writes contents to system pasteboard (`Clipboard.swift:75-110`)
3. If paste-by-default / Option-paste → `Clipboard.paste()` (`History.swift:310-325`)

### Cmd+V simulation
`Clipboard.paste()` `Clipboard.swift:113-139`:
1. `Accessibility.check()` (`114`)
2. Resolve paste key from Edit menu `NSText.paste` or fallback **Key.v + .command** (`KeyChord.swift:6-12`)
3. Build flags: `CGEventFlags` from modifiers | `0x000008` (`Clipboard.swift:118`)
4. Virtual key via **Sauce** `Sauce.shared.keyCode(for:)` with QWERTY force for command-switch layouts (`119-126`, `KeyboardLayout`)
5. `CGEventSource(stateID: .combinedSessionState)` + suppress local keyboard events (`128-131`)
6. Post keyDown + keyUp to `.cgSessionEventTap` (`133-138`)

### Accessibility
- Check: `AXIsProcessTrustedWithOptions(nil)` — `Accessibility.swift:4-10`
- `check()` currently **no-ops if not trusted** (only `guard !allowed else { return }` empty body when not allowed — still returns without prompting in this code path)
- README FAQ: user must add Maccy under **System Settings → Privacy & Security → Accessibility**
- Entitlements (`Maccy.entitlements`): App Sandbox ON, user-selected files read-only, Sparkle mach exceptions — **no special paste entitlement**; paste relies on Accessibility permission for synthetic events

Copy path without paste: just `NSPasteboard` write in `copy(_:)` (`Clipboard.swift:75-110`).
